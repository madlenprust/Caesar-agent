# Caesar 🏛️

> Автономный AI-агент для Ubuntu. Даёшь задачу — агент молча уходит работать и возвращается с готовым результатом. Без бесконечных «хочешь, продолжим?».

## Что такое Caesar

Caesar — автономный AI-агент для серверов без GUI. Идея простая: ты ставишь
задачу в Telegram или в терминале — агент берёт и делает её сам, до конца.
Не переспрашивает, не просит подтверждений на каждый чих, не бросает на середине.
Главное — экономия токенов: всё, что можно сделать без LLM, делается без неё.

**Почему так:** классические чат-боты сжигают токены на «хочешь, продолжим?» и
тащат весь контекст «на всякий случай». Caesar наоборот — тянет в LLM только
нужное, рутину закрывает готовыми «рецептами» (скиллами), а LLM зовёт только
когда реально надо подумать.

### Что умеет
- 🤖 **Автономия** — работает сам, спрашивает только когда без этого никак
- 💰 **Экономия токенов** — главный критерий архитектуры: lazy-контекст, дешёвая анализирующая LLM + умная отвечающая, скиллы
- 🧠 **Память** — факты с временной валидностью, векторный поиск по прошлым диалогам, граф сущностей
- 🔧 **Инструменты** — shell, файлы, веб-поиск, RSS, TG-каналы, документы, и т.д.
- ⏰ **Cron через разговор** — «каждый день в 9:00 делай дайджест»
- 📡 **Telegram + CLI** — общайся как удобно
- 🛡 **Self-knowledge** — агент понимает свою архитектуру и умеет безопасно расширять себя

## Установка

Одна команда (Ubuntu 22.04+, user-space, без sudo для кода):

```bash
curl -fsSL https://raw.githubusercontent.com/madlenprust/Caesar-agent/main/install.sh | bash
```

Установщик сам поставит системные зависимости, склонирует проект, создаст
виртуальное окружение, зарегистрирует systemd-сервисы и запустит setup-wizard
(там спросит LLM-ключ и токен Telegram-бота).

## Использование

### Telegram
1. Создай бота через [@BotFather](https://t.me/BotFather).
2. Запусти `caesar setup`, введи токен.
3. Напиши боту — он возьмёт задачу и вернёт результат. Прогресс виден как
   карточка с эмодзи; в финале карточка заменяется ответом.

> **Безопасность:** после установки запусти `caesar pair` — терминал покажет
> одноразовый код, ты отправишь его боту в Telegram, и бот начнёт принимать
> команды **только от тебя**. Группы — опционально (`allow_group_chats` +
> `group_access` в `config.yaml`: `owner` — только ты в группе, `all` — все).

### CLI
```bash
caesar "найди новости про Rust"   # one-shot: отправить и получить ответ
caesar                            # REPL-режим
caesar --status                   # статус демона
caesar --help                     # справка
```

### Управление
```bash
caesar setup          # переконфигурировать (LLM, TG, режим)
caesar pair           # привязать бота к твоему Telegram (одноразовый код)
caesar update         # обновить через git pull
caesar rollback       # откат к предыдущей версии
caesar status         # статус
caesar stats          # статистика по токенам
caesar doctor         # health-check системы
```

Демон (systemd user):
```bash
systemctl --user start caesar-daemon
systemctl --user stop caesar-daemon
systemctl --user restart caesar-daemon
systemctl --user status caesar-daemon
journalctl --user -u caesar-daemon -f       # смотреть логи
```

### Cron через разговор
```
Ты: «Каждый день в 9:00 делай дайджест новостей»

Caesar: 📅 Понял так:
        - Когда: каждый день в 09:00
        - Что: дайджест новостей
        Всё верно? [Да / Нет]
```
Понимает: «каждый день в N», «по будням», «каждый понедельник»,
«каждые N минут/часов», «раз в неделю», «каждое утро/вечер».
В «тихие часы» уведомления переносятся — ночью тебя не дёргают.
